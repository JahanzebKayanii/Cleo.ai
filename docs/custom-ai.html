FAQ

Common questions

How do you price projects?
We quote a fixed price after understanding your requirements. No hourly billing, no scope creep surprises. The price we quote is the price you pay.
What stack do you build on?
Typically Python (FastAPI) + React, with PostgreSQL, Claude for AI, Deepgram for speech, and Twilio for voice. We can work with your existing stack if needed.
Do I own the code?
Yes. Full source code handover is included in every project. You own everything we build, no strings attached.
What if I need changes after launch?
A 30-day post-launch support window is included in every project. After that, we offer retainer arrangements for ongoing changes and improvements.
